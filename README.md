## 📝 Task 1: To-Do List Application

A simple yet effective *To-Do List App* to help users manage their daily tasks. Users can add, edit, complete, and delete tasks — all changes are saved locally for persistence.

### 📌 Features
- Built using Tkinter GUI  
- Add, edit, mark complete, delete tasks  
- Tasks stored in a local JSON file  
- Clean interface with automatic updates

### 📂 File Structure

📁 To-Do-List/
├── todo_app.py
├── tasks.json


### ▶ How to Run
bash
python todo_app.py

## 🧮 Task 2: Advanced GUI Calculator

A visually interactive *Calculator App* designed to perform both basic and advanced operations. The UI guides users to select operations first, then input values accordingly.

- User-friendly button layout

### 📂 File Structure

├── calculator_app.py

## 🔐 Task 3: Password Generator with Master Security

An all-in-one *Password Generator* tool with multiple character options, strength indication, clipboard support, and secured access to saved passwords using a *master password*.

### 📌 Features
- Select character types (A-Z, a-z, 0-9, symbols)  
- Set desired password length  
- Strength meter: Weak / Strong  
- Save passwords with labels in a .txt file  
- Master password protection to view/change saved entries  
- Change master password securely  
- Clipboard copy functionality

### 📂 File Structure

📁 Password-Generator/
├── password_generator.py
├── saved_passwords.txt
├── master.txt


### ▶ How to Run
bash
python password_generator.py
